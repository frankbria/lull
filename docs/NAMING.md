# Naming & Domain — research record

**Status:** DEFERRED. Working name = **"Lull"** (repo + internal). Final public name to be chosen at
marketing time (before Sprint 6 / store submission). This file preserves the research so we don't
repeat it.

## Constraints
- **Store-safe:** no "ketamine," "psychedelic," or drug-facilitation language anywhere user/store-facing.
- Evoke a calm, guided descent into an altered/trance/liminal state; pronounceable; ownable.
- Avoid existing trademarks in wellness/sleep/audio (e.g. **Lull** = mattress brand, **Calm**,
  **Headspace**, **Eventide** = audio plugins, **Cadence** = health co, **Reverie** = adjustable beds).

## Key finding
Every clean single-word `.com`/`.app` for popular wellness words is **squatted or trademarked**
(reverie, selah, somni, numen, liminal, limen, oneira, otium, etc. — all taken). **Coined names are
the only ones with the exact `.com` still free.**

## Lead candidate explored: blends of **Lull + Limen** ("calm" + "threshold of consciousness")
Domain availability checked via Verisign/RDAP (2026-06-20):

| Name | `.com` | Notes |
|---|---|---|
| **Lulmen** | ✅ free | Shortest/most elegant; **closest to "Lumen"** → highest collision |
| **Lullmen** | ✅ free | Double-L surfaces "Lull", reads less like Lumen; still "-men" ending |
| **Lulimen** | ✅ free | "lim" surfaces "Limen"; best Lumen-escape of the close blends |
| **Lulmena** | ✅ free | "-a" drops the masculine "-men" read; most distinctive |
| **Lullimen** | ✅ free | Clearest blend but longest |

Also free: `lulmen.io`, `lullmen.com`. Taken: lulim, lullen, lumin, lullam.

## The "Lumen" problem (why this was deferred, not locked)
Search engines auto-correct **"Lulmen" → "Lumen"**, and the Lumen field is crowded *and adjacent*:
- **Lumen (lumen.me)** — $62M-funded wellness/metabolism **app** (direct space overlap → discoverability + trademark-confusion risk)
- **Lumen Technologies** (telecom), **Lumen** (over-50 dating app)

→ Any Lull+Limen blend inherits some of this. Acceptable for a founder willing to coexist; flagged
for trademark counsel before locking.

## Decision
Park naming. Keep "Lull" internally. When we resume: weigh the blend candidates above vs. a larger
swerve away from "Lumen," run a trademark (USPTO/class 9 + 44) and App Store/Play name search, and
secure the `.com` + social handles in one pass.
